Info

HomeFinder was created as a final project in the technological specialization course in information systems and programming at IEFP together with two colleagues (Dario Bianchi and Jos√© Lebre). The project was built in two phases, where in the first phase we built the database from scratch and the front pages where the user can search and register their properties and in the second phase we built the dashboard where users can manage all aspects related to their properties and rentals.

The Challenge

Managing all the aspects related to a rental can be quite a complicated and confusing process. Losing documents is quite common when dealing with the process of a rental.

The Solution

HomeFinder allows you to advertise properties and at the same time do and track all the internal management of the rentals.
